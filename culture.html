Meet the people.

Discover the characters that give Web world it's distinct personality.

HTML, otherwise known as Hypertext Markup Language, is the structure of every region in Web-World. All the pieces, or content, of web world can be found in HTML. It is not much to look at it but it is robust, to the point and really has everything you will need to make up a region of web-world.

See for yourself!

Press shift + Ctrl + i ( For PC users ) || press shift + Command + i ( For Mac users )

Alternatively right click + select "Inspect"

What do you think?...It is pretty but without it there would be no web-world!

a links A very social character keeping everyone in web-world connected. links are constructed and thus found anywhere two points that might not otherwise to connected, just like a bridge.

CSS, otherwise known as Cascading Style Sheets are what enables web-world to be so multifacted, multicolored and stylistically unique. All the regions, or content, of web world are given there aesthetic voice through CSS. The region of CSS is a place where colors, fonts, layouts (all the shiny things) live. At times CSS can be finicky and difficult to wrangle but once you start understanding it's particular language it becomes a lot smoother to navigate.

See it for yourself!

  1. Press shift + Ctrl + i ( For PC users ) || press shift + Command + i ( For Mac users )
  2. Alternatively right click + select "Inspect"
  3. Inside the "Elements" tab find "head" tag and make sure it is open by clicking the down caret
  4. Inside the head tage navigate to the linked stylesheet
  5. right Click + select "Hide Element"
  6. All the styles should disappear!
  7. Don't be afraid!
  8. Refresh the page to restore all the styles

/* */ Comments Very talkative and keeps everyone informed. Found between style changes in web-world regions.
{ } Curly Braces Encapsulating and warm, Curly Braces wrap style properties given to web-world regions. Found around all styles.
; Semi-colon Definitive and to the point the Semi-colon defines one style property over another. Found at the end of each style property.
: Forward Slash The Colon, not to be confused with the semi-color, is an inherent Mediator as it decides the degree to which a style will be applied depending on context. Found between Style properties and their assigned value.
. Period Unassuming, easily missed but easily to bridge HTML to style properties Found at the beginning of any style decision to connect HTML regions to their aesthetic suit.

When web world was first created HTML was alone. Web World did not have much of a personality. It was hard to look at but it worked. The issue of establishing a global personality was addressed by the creation of CSS a few years later. As CSS was created specifically for HTMl it goes without saying (Although I feel obliged to say it) that they work peacefully together to create what Web-World has become today!
